This paper introduces the concept of a 'Jordan-segmentable mask,' a topology-aware definition for binary image segmentation evaluation. It proposes a mathematically rigorous, unsupervised criterion based on the Jordan Curve Theorem, digital topology, and homology theory to assess whether a segmentation correctly partitions an image into two topologically separate, connected regions. üìÖ 2026-01-15 ‚≠ê 0.70 üìÇ cs.CL
Authors: Syed Naveed Mahmood, Md. Rezaur Rahman Bhuiyan, Tasfia Zaman et al.
This paper introduces the Knowledge Immunization Framework (KIF), a representation-aware unlearning method for LLMs that targets internal activation signatures to achieve true knowledge erasure, differentiating it from mere behavioral suppression. KIF successfully breaks the stability-erasure tradeoff seen in prior work, demonstrating near-oracle erasure (FQ ~0.99) while preserving model utility (MU = 0.62) across various LLMs from 3B to 14B parameters. The framework provides the first systematic diagnosis of mechanism-level forgetting behavior.

Authors: Harry Aytug
The causalfe package introduces a Python implementation of Causal Forests with Fixed Effects (CFFE) to accurately estimate heterogeneous treatment effects in panel data settings.
