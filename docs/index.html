Authors: Shahnawaz Alam, Mohammed Mudassir Uddin, Mohammed Kaif Pasha
This paper addresses the dual challenge of deploying accurate deep learning models for plant disease detection on low-cost edge devices and learning from scarce labeled data in remote agricultural areas. It proposes a Prune-then-Meta-Learn-then-Prune (PMP) pipeline, integrating Disease-Aware Channel Importance Scoring (DACIS), to combine neural network pruning with few-shot learning. This approach achieves a 78% model size reduction while retaining 92.3% of the original accuracy, enabling real-time field diagnosis at 7 frames per second on a Raspberry Pi 4.
