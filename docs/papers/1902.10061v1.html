Supervised learning improves disease outbreak detection

arXiv ID: 1902.10061v1
Published: 2019-02-06
Authors: Benedikt Zacher, Irina Czogiel
Categories: stat.AP, cs.LG, stat.ML
Relevance Score: 1.00 / 1.00
View on arXiv Download PDF

Summary

This paper introduces the first supervised learning approach for infectious disease outbreak detection, leveraging Hidden Markov Models (HMMs) and routinely collected public health surveillance data. The proposed model significantly reduces false positive rates by up to 50% while retaining equivalent sensitivity compared to a state-of-the-art statistical method. This advancement aims to improve the efficiency and accuracy of public health surveillance systems.

Medical Relevance

Early and accurate detection of infectious disease outbreaks is paramount for implementing timely public health interventions, controlling disease spread, and protecting population health from epidemics and pandemics.

AI Health Application

The AI application involves developing a supervised learning model, specifically using Hidden Markov Models, to improve the accuracy and reduce false positives in the detection of infectious disease outbreaks from routine public health surveillance data. This aims to enhance the efficiency and effectiveness of early warning systems for public health.

Key Points

  • Develops the first supervised learning approach based on Hidden Markov Models (HMMs) specifically for disease outbreak detection.
  • Utilizes routinely collected data within established public health surveillance systems.
  • Evaluates the model's performance using real-world infectious disease data (Salmonella and Campylobacter) and simulated scenarios.
  • Compares the proposed method against a state-of-the-art statistical approach currently deployed in multiple European countries (e.g., Germany).
  • Demonstrates a significant reduction in the false positive rate by up to 50% compared to the benchmark.
  • Maintains the same level of sensitivity as the established state-of-the-art outbreak detection method.
  • Positions supervised learning as a crucial next step for advancing machine learning applications in public health surveillance.

Methodology

The study proposes a supervised learning approach built upon Hidden Markov Models (HMMs) for detecting infectious disease outbreaks. This model is trained using routinely collected surveillance data. Its efficacy is evaluated through experiments on real Salmonella and Campylobacter incidence data, as well as simulated datasets. Performance is benchmarked against a widely used, state-of-the-art statistical method for outbreak detection.

Key Findings

The supervised learning HMM model achieved a reduction in the false positive rate by up to 50% when compared to a state-of-the-art outbreak detection approach, while concurrently maintaining an equivalent level of sensitivity.

Clinical Impact

By significantly reducing false positives, this approach can alleviate the workload on public health professionals, allowing them to allocate resources more effectively to genuine outbreak investigations. This leads to more efficient surveillance systems and potentially faster, more targeted public health responses, ultimately improving disease control and population health outcomes.

Limitations

Future Directions

The authors advocate for continued development of machine learning applications for disease outbreak detection, seeing their supervised learning approach as a foundational step towards improving future public health surveillance systems.

Abstract

The early detection of infectious disease outbreaks is a crucial task to protect population health. To this end, public health surveillance systems have been established to systematically collect and analyse infectious disease data. A variety of statistical tools are available, which detect potential outbreaks as abberations from an expected endemic level using these data. Here, we develop the first supervised learning approach based on hidden Markov models for disease outbreak detection, which leverages data that is routinely collected within a public health surveillance system. We evaluate our model using real Salmonella and Campylobacter data, as well as simulations. In comparison to a state-of-the-art approach, which is applied in multiple European countries including Germany, our proposed model reduces the false positive rate by up to 50% while retaining the same sensitivity. We see our supervised learning approach as a significant step to further develop machine learning applications for disease outbreak detection, which will be instrumental to improve public health surveillance systems.
