A Hybrid Deep Learning Framework with Explainable AI for Lung Cancer Classification with DenseNet169 and SVM

arXiv ID: 2512.03359v1
Published: 2025-12-03
Authors: Md Rashidul Islam, Bakary Gibba, Altagi Abdallah Bakheit Abdelgadir
Categories: cs.CV
Relevance Score: 1.00 / 1.00
View on arXiv Download PDF

Summary

This paper proposes a hybrid deep learning framework with explainable AI for automated lung cancer classification using CT scans. The system leverages DenseNet169 with attention mechanisms and multi-scale feature fusion, alongside an SVM model with MobileNetV2 features, achieving 98% accuracy for classifying Normal, Benign, and Malignant cases, while providing crucial interpretability.

Medical Relevance

Early and accurate diagnosis of lung cancer is paramount for improving patient survival rates. This research directly addresses this by providing an automated, highly accurate, and interpretable system for classifying lung CT scans, potentially reducing diagnostic delays and human error in a deadly disease.

AI Health Application

The AI application described is an automatic lung cancer classification system that uses deep learning (DenseNet169, MobileNetV2, SVM) to analyze CT scans and categorize cases as Normal, Benign, or Malignant. It integrates Explainable AI (Grad-CAM, SHAP) to provide transparency into the diagnostic decisions, making it a valuable tool to assist radiologists and clinicians in early and accurate lung cancer detection.

Key Points

  • Addresses the critical need for early and accurate lung cancer diagnosis, overcoming the limitations of manual CT scan interpretation (time-consuming, prone to human error).
  • Proposes a hybrid deep learning framework utilizing the public IQOTHNCCD CT scan dataset, categorized into Normal, Benign, and Malignant lung conditions.
  • Implements a DenseNet169 model enhanced with Squeeze-and-Excitation (SE) blocks for attention-based feature extraction, Focal Loss to manage class imbalance, and a Feature Pyramid Network (FPN) for multi-scale feature fusion.
  • Develops a second classification pathway using a Support Vector Machine (SVM) model, which extracts features via MobileNetV2 to enhance its performance.
  • Integrates Explainable AI (XAI) techniques: Grad-CAM for visualizing decision-making regions in CT scans for the DenseNet169 model, and SHAP (Shapley Additive Explanations) for explaining feature contributions within the SVM model.
  • Achieved a high classification accuracy of 98% for both the DenseNet169 and SVM models during intensive evaluation.
  • The high accuracy combined with robust interpretability suggests significant potential for deployment in real-world clinical settings to improve lung cancer diagnosis.

Methodology

The study employs a hybrid deep learning framework. The primary model is DenseNet169, augmented with Squeeze-and-Excitation (SE) blocks for attention, Focal Loss to handle class imbalance, and a Feature Pyramid Network (FPN) for multi-scale feature fusion. A secondary model is an SVM, which uses MobileNetV2 for feature extraction. Both models were trained and evaluated on the public IQOTHNCCD CT scan dataset (Normal, Benign, Malignant). Explainable AI techniques, Grad-CAM, were used for DenseNet169 to visualize decision regions, and SHAP was used for the SVM model to explain feature contributions.

Key Findings

Both the DenseNet169-based deep learning model and the SVM model (with MobileNetV2 features) achieved an impressive 98% classification accuracy for differentiating Normal, Benign, and Malignant lung cancer cases from CT scans, demonstrating high robustness and reliability.

Clinical Impact

This automated system has the potential to significantly enhance lung cancer diagnosis by providing clinicians with highly accurate and interpretable classifications of CT scans. It could reduce diagnostic workload, minimize human error, and facilitate earlier intervention, ultimately leading to improved patient outcomes and survival rates in oncology practice.

Abstract

Lung cancer is a very deadly disease worldwide, and its early diagnosis is crucial for increasing patient survival rates. Computed tomography (CT) scans are widely used for lung cancer diagnosis as they can give detailed lung structures. However, manual interpretation is time-consuming and prone to human error. To surmount this challenge, the study proposes a deep learning-based automatic lung cancer classification system to enhance detection accuracy and interpretability. The IQOTHNCCD lung cancer dataset is utilized, which is a public CT scan dataset consisting of cases categorized into Normal, Benign, and Malignant and used DenseNet169, which includes Squeezeand-Excitation blocks for attention-based feature extraction, Focal Loss for handling class imbalance, and a Feature Pyramid Network (FPN) for multi-scale feature fusion. In addition, an SVM model was developed using MobileNetV2 for feature extraction, improving its classification performance. For model interpretability enhancement, the study integrated Grad-CAM for the visualization of decision-making regions in CT scans and SHAP (Shapley Additive Explanations) for explanation of feature contributions within the SVM model. Intensive evaluation was performed, and it was found that both DenseNet169 and SVM models achieved 98% accuracy, suggesting their robustness for real-world medical practice. These results open up the potential for deep learning to improve the diagnosis of lung cancer by a higher level of accuracy, transparency, and robustness.
