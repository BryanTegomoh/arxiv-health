UnifiedFL: A Dynamic Unified Learning Framework for Equitable Federation

Published: 2025-10-30
Authors: Furkan Pala, Islem Rekik
Categories: cs.LG
Relevance Score: 1.00 / 1.00

Summary

UnifiedFL is a novel federated learning (FL) framework designed to overcome critical challenges where clients possess fundamentally different neural architectures and non-identically distributed datasets with significant domain shifts. It unifies heterogeneous local networks using a shared Graph Neural Network (GNN), distance-driven clustering, and a two-tier aggregation policy to achieve superior performance in medical image classification and segmentation tasks.

Medical Relevance

This framework significantly enhances the applicability of privacy-preserving federated learning in healthcare by enabling collaboration across diverse medical institutions that may use different AI models and have varied patient populations or data acquisition methods, directly benefiting fields like radiology and pathology.

AI Health Application

This research contributes to the development of more robust and equitable Federated Learning frameworks for healthcare. Specifically, it enables collaborative AI model training across different medical institutions (e.g., hospitals, research labs) without sharing raw patient data, thereby protecting privacy. This can lead to improved AI models for tasks like medical image classification (e.g., MedMNIST datasets for various diagnostic tasks) and segmentation (e.g., hippocampus segmentation for neurological disease assessment), even when institutions use different AI architectures or have diverse patient populations/data distributions.

Key Points

  • Addresses architectural heterogeneity by accommodating fundamentally different network types (e.g., CNNs, GNNs, MLPs), unlike prior methods limited to architectural variants within a single model family.
  • Tackles the 'domain-fracture problem' where client data distributions differ markedly from testing environments, improving overall model generalizability.
  • Proposes representing heterogeneous local networks as nodes and edges in a directed model graph, which is a novel structural approach for FL.
  • Employs a common Graph Neural Network (GNN) to parameterize and unify all diverse client architectures within the federated system.
  • Introduces distance-driven clustering of clients based on Euclidean distances between their network parameters to group similar models.
  • Implements a two-tier aggregation policy strategically designed to balance model convergence speed with diversity across the federated network.
  • Demonstrates superior performance on critical medical imaging benchmarks, including MedMNIST classification and hippocampus segmentation.

Methodology

UnifiedFL represents heterogeneous local networks as nodes and edges within a directed model graph. A shared Graph Neural Network (GNN) is used to parameterize all diverse client architectures. The framework incorporates distance-driven clustering, grouping clients based on Euclidean distances calculated from their network parameters. Furthermore, it employs a two-tier aggregation policy designed to balance the trade-off between model convergence and diversity across the federated clients.

Key Findings

UnifiedFL demonstrated superior performance in experiments conducted on MedMNIST classification and hippocampus segmentation benchmarks. This indicates its effectiveness in handling complex federated scenarios involving diverse client architectures, non-identically distributed data, and distinct test domains, outperforming existing FL methods.

Clinical Impact

UnifiedFL can foster more robust and equitable AI model development in clinical settings. It allows hospitals and research centers with varied data sources, acquisition protocols, and computational infrastructures (manifested in different local model architectures) to collaboratively train powerful AI models without compromising data privacy. This could lead to more generalizable and clinically useful diagnostic, prognostic, and analytical tools in medical imaging across a broader range of real-world patient populations.

Limitations

The abstract primarily highlights the significant limitations of *existing* federated learning frameworks that UnifiedFL aims to overcome. These include their inability to tolerate fundamentally different network types (beyond architectural variants), their failure to adequately address the 'domain-fracture problem' where client data distributions differ from testing domains, and their poor performance under combined architectural and statistical heterogeneity with domain shifts. The abstract does not explicitly state limitations or caveats for UnifiedFL itself.

Abstract

Federated learning (FL) has emerged as a key paradigm for collaborative model training across multiple clients without sharing raw data, enabling privacy-preserving applications in areas such as radiology and pathology. However, works on collaborative training across clients with fundamentally different neural architectures and non-identically distributed datasets remain scarce. Existing FL frameworks face several limitations. Despite claiming to support architectural heterogeneity, most recent FL methods only tolerate variants within a single model family (e.g., shallower, deeper, or wider CNNs), still presuming a shared global architecture and failing to accommodate federations where clients deploy fundamentally different network types (e.g., CNNs, GNNs, MLPs). Moreover, existing approaches often address only statistical heterogeneity while overlooking the domain-fracture problem, where each client's data distribution differs markedly from that faced at testing time, undermining model generalizability. When clients use different architectures, have non-identically distributed data, and encounter distinct test domains, current methods perform poorly. To address these challenges, we propose UnifiedFL, a dynamic federated learning framework that represents heterogeneous local networks as nodes and edges in a directed model graph optimized by a shared graph neural network (GNN). UnifiedFL introduces (i) a common GNN to parameterize all architectures, (ii) distance-driven clustering via Euclidean distances between clients' parameters, and (iii) a two-tier aggregation policy balancing convergence and diversity. Experiments on MedMNIST classification and hippocampus segmentation benchmarks demonstrate UnifiedFL's superior performance. Code and data: https://github.com/basiralab/UnifiedFL
