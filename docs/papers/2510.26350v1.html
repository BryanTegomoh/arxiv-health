UnifiedFL: A Dynamic Unified Learning Framework for Equitable Federation

arXiv ID: 2510.26350v1
Published: 2025-10-30
Authors: Furkan Pala, Islem Rekik
Categories: cs.LG
Relevance Score: 1.00 / 1.00
View on arXiv Download PDF

Summary

UnifiedFL introduces a novel dynamic federated learning framework designed to overcome significant limitations in existing FL approaches, particularly architectural heterogeneity, statistical heterogeneity, and domain-fracture across clients. It achieves this by representing diverse local neural networks as a directed model graph optimized by a shared graph neural network (GNN), incorporating distance-driven clustering and a two-tier aggregation policy. Experiments on MedMNIST and hippocampus segmentation benchmarks demonstrate UnifiedFL's superior performance in fostering equitable federation.

Medical Relevance

This framework is highly relevant for medical/health research as it facilitates privacy-preserving collaborative AI model training across diverse healthcare institutions. It addresses the real-world challenge where different hospitals or clinics may possess fundamentally different data types, patient demographics, and computational infrastructure, enabling more robust and generalizable AI applications in fields like radiology and pathology without raw data sharing.

AI Health Application

This research develops an advanced Federated Learning (FL) framework, UnifiedFL, designed to facilitate privacy-preserving collaborative AI model training across multiple healthcare clients. It specifically addresses challenges arising from fundamentally different neural architectures and diverse patient data distributions (statistical and domain heterogeneity) among clients. By enabling robust model training in such heterogeneous environments, UnifiedFL can be applied to develop more generalizable AI models for tasks like medical image classification (e.g., MedMNIST datasets) and segmentation (e.g., hippocampus segmentation for neurological assessment) in areas like radiology and pathology, without requiring the direct sharing of sensitive patient data.

Key Points

  • Addresses the critical gaps in federated learning where existing methods fail to accommodate fundamentally different neural architectures (e.g., CNNs, GNNs, MLPs) and robustly handle statistical heterogeneity and domain-fracture.
  • Proposes UnifiedFL, a dynamic FL framework that represents all heterogeneous local networks as nodes and edges within a directed model graph.
  • Introduces a common Graph Neural Network (GNN) that effectively parameterizes and unifies diverse architectures across federated clients.
  • Employs a distance-driven clustering mechanism, utilizing Euclidean distances between clients' parameters to group similar models.
  • Implements a two-tier aggregation policy designed to strike a balance between model convergence and client diversity.
  • Demonstrates superior performance on medical benchmarks, specifically MedMNIST classification and hippocampus segmentation tasks, showcasing its practical applicability in healthcare.
  • Aims to enable more equitable federation by allowing clients with highly varied computational resources, data distributions, and model preferences to collaboratively train a robust global model.

Methodology

UnifiedFL proposes a dynamic federated learning framework. It conceptualizes heterogeneous local networks as nodes and edges within a directed model graph. A shared Graph Neural Network (GNN) is then used to parameterize and optimize all these diverse architectures. The framework incorporates distance-driven clustering, based on Euclidean distances between client parameters, and employs a two-tier aggregation policy to manage the balance between convergence and diversity.

Key Findings

UnifiedFL consistently outperforms existing federated learning methods, particularly in scenarios involving significant architectural and statistical heterogeneity, alongside domain-fracture. Its effectiveness was empirically validated through superior results on standard medical benchmarks, specifically MedMNIST for classification and hippocampus segmentation tasks.

Clinical Impact

UnifiedFL has the potential to significantly advance collaborative AI development in clinical settings. By enabling hospitals and clinics with varying data distributions, hardware, and preferred neural network architectures to contribute to a unified model, it can lead to more robust, generalizable, and ethically developed medical AI tools. This could improve diagnostic accuracy, treatment planning, and personalized medicine across diverse patient populations, ultimately enhancing patient care by leveraging distributed medical data more effectively and equitably.

Abstract

Federated learning (FL) has emerged as a key paradigm for collaborative model training across multiple clients without sharing raw data, enabling privacy-preserving applications in areas such as radiology and pathology. However, works on collaborative training across clients with fundamentally different neural architectures and non-identically distributed datasets remain scarce. Existing FL frameworks face several limitations. Despite claiming to support architectural heterogeneity, most recent FL methods only tolerate variants within a single model family (e.g., shallower, deeper, or wider CNNs), still presuming a shared global architecture and failing to accommodate federations where clients deploy fundamentally different network types (e.g., CNNs, GNNs, MLPs). Moreover, existing approaches often address only statistical heterogeneity while overlooking the domain-fracture problem, where each client's data distribution differs markedly from that faced at testing time, undermining model generalizability. When clients use different architectures, have non-identically distributed data, and encounter distinct test domains, current methods perform poorly. To address these challenges, we propose UnifiedFL, a dynamic federated learning framework that represents heterogeneous local networks as nodes and edges in a directed model graph optimized by a shared graph neural network (GNN). UnifiedFL introduces (i) a common GNN to parameterize all architectures, (ii) distance-driven clustering via Euclidean distances between clients' parameters, and (iii) a two-tier aggregation policy balancing convergence and diversity. Experiments on MedMNIST classification and hippocampus segmentation benchmarks demonstrate UnifiedFL's superior performance. Code and data: https://github.com/basiralab/UnifiedFL
