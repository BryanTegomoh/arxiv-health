WSD-MIL: Window Scale Decay Multiple Instance Learning for Whole Slide Image Classification

arXiv ID: 2512.19982v1
Published: 2025-12-23
Authors: Le Feng, Li Xiao
Categories: cs.CV
Relevance Score: 1.00 / 1.00

Summary

WSD-MIL proposes a novel Multiple Instance Learning (MIL) approach for Whole Slide Image (WSI) classification that addresses the limitations of existing Transformer-based methods regarding computational complexity and varying tumor scales. It introduces a window scale decay based attention module for efficient multi-scale local relationship modeling and a region gate module for enhanced global information, achieving state-of-the-art performance with significant memory reduction on cancer datasets.

Medical Relevance

This research significantly enhances the accuracy and efficiency of computational pathology for cancer diagnosis by providing a more robust and scalable method for analyzing Whole Slide Images, directly contributing to improved diagnostic tools for pathologists.

AI Health Application

Developing advanced AI models (specifically Multiple Instance Learning with novel attention mechanisms) for automated analysis and classification of whole slide pathology images, primarily to detect and characterize tumor regions for cancer diagnosis. This enhances diagnostic accuracy and efficiency in computational pathology workflows.

Key Points

  • Existing Transformer-based MIL methods for WSI classification face challenges due to quadratic computational complexity, limiting scalability to large datasets.
  • Previous methods struggle with precisely capturing local instance correlations due to fixed-scale attention mechanisms, failing to account for varying tumor region scales and the distance-based decay effect of patch relevance.
  • WSD-MIL introduces a 'window scale decay based attention module' that uses a cluster-based sampling strategy to reduce computational costs while progressively decaying attention window-scales to model local instance relationships at varying scales.
  • It also incorporates a 'squeeze-and-excitation based region gate module' designed to dynamically adjust window weights, thereby enhancing global information modeling within the WSI.
  • The proposed WSD-MIL achieves state-of-the-art diagnostic accuracy on two prominent cancer datasets: CAMELYON16 (breast cancer metastasis detection) and TCGA-BRCA (breast cancer classification).
  • A significant practical advantage is the reduction of computational memory by 62%, making WSD-MIL more efficient and scalable for analyzing large-scale WSIs.
  • The authors commit to making the code publicly available, promoting reproducibility and further development in the computational pathology community.

Methodology

WSD-MIL is a deep learning framework built upon the Multiple Instance Learning (MIL) paradigm. Its core components are a window scale decay based attention module, which employs cluster-based sampling and progressively decaying attention window-scales to capture multi-scale local instance relationships efficiently, and a squeeze-and-excitation based region gate module, which dynamically adjusts window weights to enhance global information modeling.

Key Findings

WSD-MIL achieved state-of-the-art performance on the CAMELYON16 and TCGA-BRCA datasets for WSI classification. A major finding is its significant computational efficiency improvement, demonstrated by a 62% reduction in memory usage compared to existing methods, while maintaining or improving diagnostic accuracy.

Clinical Impact

By improving the accuracy and computational efficiency of WSI analysis, WSD-MIL has the potential to accelerate and enhance cancer diagnosis in clinical pathology. More precise identification of tumor regions, especially those with varying scales, can lead to more accurate prognoses and treatment decisions, while reduced memory footprint allows for wider deployment and faster processing in clinical workflows.

Abstract

In recent years, the integration of pre-trained foundational models with multiple instance learning (MIL) has improved diagnostic accuracy in computational pathology. However, existing MIL methods focus on optimizing feature extractors and aggregation strategies while overlooking the complex semantic relationships among instances within whole slide image (WSI). Although Transformer-based MIL approaches aiming to model instance dependencies, the quadratic computational complexity limits their scalability to large-scale WSIs. Moreover, due to the pronounced variations in tumor region scales across different WSIs, existing Transformer-based methods employing fixed-scale attention mechanisms face significant challenges in precisely capturing local instance correlations and fail to account for the distance-based decay effect of patch relevance. To address these challenges, we propose window scale decay MIL (WSD-MIL), designed to enhance the capacity to model tumor regions of varying scales while improving computational efficiency. WSD-MIL comprises: 1) a window scale decay based attention module, which employs a cluster-based sampling strategy to reduce computational costs while progressively decaying attention window-scale to capture local instance relationships at varying scales; and 2) a squeeze-and-excitation based region gate module, which dynamically adjusts window weights to enhance global information modeling. Experimental results demonstrate that WSD-MIL achieves state-of-the-art performance on the CAMELYON16 and TCGA-BRCA datasets while reducing 62% of the computational memory. The code will be publicly available.
