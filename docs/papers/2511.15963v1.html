PySERA: Open-Source Standardized Python Library for Automated, Scalable, and Reproducible Handcrafted and Deep Radiomics

Summary

PySERA is an open-source, Python-native library designed to standardize and automate both handcrafted and deep learning radiomics, addressing issues of reproducibility and scalability in medical image analysis. It provides a robust framework for extracting quantitative biomarkers from medical images, integrating seamlessly with AI ecosystems, and demonstrating superior reproducibility and predictive accuracy compared to existing tools.

Medical Relevance

By providing a standardized, reproducible, and scalable platform for extracting quantitative biomarkers from medical images, PySERA directly supports the development of precision medicine and AI-driven diagnostics. It enables more reliable research and clinical translation of imaging biomarkers for disease characterization, prognosis, and treatment response prediction.

AI Health Application

PySERA provides an open-source, standardized Python library for automated and reproducible extraction of both handcrafted and deep learning (AI) based radiomics features from medical images. These features serve as quantitative biomarkers for precision modeling in healthcare, enabling the development and validation of AI models for improved diagnosis, prognosis, and treatment planning. It facilitates 'AI ready imaging research' by standardizing data preprocessing and feature extraction for machine learning and deep learning applications in medicine.

Key Points

  • PySERA is an open-source Python library reimplementing the MATLAB-based SERA platform, offering a modular and object-oriented architecture for radiomics analysis.
  • It supports extraction of 557 features, including 487 Image Biomarker Standardization Initiative (IBSI) compliant features, 10 moment invariants, 60 diagnostic features, and deep learning radiomics embeddings from pretrained networks (ResNet50, DenseNet121, VGG16).
  • The framework includes standardized preprocessing (resampling, discretization, normalization), multi-format image I/O, adaptive memory management, and a parallel multicore extraction engine for scalability.
  • PySERA integrates natively with major machine learning ecosystems such as scikit-learn, PyTorch, TensorFlow, and MONAI, facilitating seamless AI model development and deployment.
  • In IBSI benchmarks, PySERA achieved over 94% reproducibility, outperforming PyRadiomics and closely matching MITK, indicating high reliability and standardization.
  • Across eight public datasets, PySERA demonstrated predictive accuracies ranging from 0.54 to 0.87, consistently exceeding the performance of PyRadiomics.
  • This tool unifies handcrafted and deep learning radiomics in a transparent, scalable, and extensible Python framework, establishing a robust foundation for reproducible, AI-ready imaging research.

Methodology

PySERA was developed as an open-source, Python-native, object-oriented library. It reimplements the feature extraction capabilities of the MATLAB-based SERA platform, incorporating both handcrafted features (IBSI compliant, moment invariants, diagnostic) and deep learning radiomics embeddings from pre-trained networks (ResNet50, DenseNet121, VGG16). The methodology includes standardized preprocessing, multi-format I/O, adaptive memory management, and parallel processing. Validation involved IBSI benchmarks for reproducibility and comparative analysis against PyRadiomics and MITK, alongside predictive accuracy evaluations on eight public datasets for clinical utility.

Key Findings

PySERA achieved over 94% reproducibility in IBSI benchmarks, demonstrating superior standardization compared to PyRadiomics and strong alignment with MITK. It consistently outperformed PyRadiomics in predictive accuracy across eight public datasets, with accuracies ranging from 0.54 to 0.87. The framework successfully unifies handcrafted and deep learning radiomics within a single, scalable, and extensible Python environment.

Its integration with AI ecosystems facilitates the development of robust predictive models for diagnosis, prognosis, and treatment monitoring, thereby enhancing precision medicine and supporting evidence-based clinical decision-making across diverse medical specialties that rely on medical imaging.

Abstract

Radiomics enables the extraction of quantitative biomarkers from medical images for precision modeling, but reproducibility and scalability remain limited due to heterogeneous software implementations and incomplete adherence to standards. Existing tools also lack unified support for deep learning based radiomics. To address these limitations, we introduce PySERA, an open source, Python native, standardized radiomics framework designed for automation, reproducibility, and seamless AI integration. PySERA reimplements the MATLAB based SERA platform within a modular, object oriented architecture and computes 557 features, including 487 Image Biomarker Standardization Initiative (IBSI) compliant features, 10 moment invariant descriptors, and 60 diagnostic features, together with deep learning radiomics embeddings from pretrained networks such as ResNet50, DenseNet121, and VGG16. The framework provides standardized preprocessing, including resampling, discretization, and normalization, multi format image input and output, adaptive memory management, and a parallel multicores extraction engine. PySERA integrates natively with major machine learning ecosystems including scikit learn, PyTorch, TensorFlow, MONAI and others. In IBSI benchmarks, PySERA achieved more than 94 percent reproducibility and outperformed PyRadiomics while closely matching MITK. Across eight public datasets, it achieved predictive accuracies ranging from 0.54 to 0.87, consistently exceeding PyRadiomics. PySERA unifies handcrafted and deep learning radiomics in a transparent, scalable, and extensible Python framework, establishing a robust foundation for reproducible, AI ready imaging research.
