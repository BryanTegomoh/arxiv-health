PySERA: Open-Source Standardized Python Library for Automated, Scalable, and Reproducible Handcrafted and Deep Radiomics

arXiv ID: 2511.15963v1
Published: 2025-11-20
Authors: Mohammad R. Salmanpour, Amir Hossein Pouria, Sirwan Barichin, Yasaman Salehi, Sonya Falahati, Isaac Shiri, Mehrdad Oveisi, Arman Rahmim

Summary

PySERA is an open-source Python library designed to enhance reproducibility, scalability, and AI integration in radiomics by unifying handcrafted and deep learning feature extraction. It achieves over 94% reproducibility in IBSI benchmarks and demonstrates superior predictive accuracy compared to existing tools, providing a robust foundation for AI-ready medical imaging research.

Medical Relevance

This framework is critical for advancing precision medicine by enabling the extraction of reproducible and scalable quantitative biomarkers from medical images, which are essential for developing robust AI-powered diagnostic, prognostic, and predictive models.

AI Health Application

PySERA is a foundational AI health application. It provides a standardized, scalable, and reproducible Python library for extracting both handcrafted and deep learning features from medical images. This enables the development and deployment of AI models for medical diagnosis, prognosis, treatment response prediction, and other precision medicine applications by providing the necessary input features for machine learning algorithms. It directly supports AI-driven medical imaging research and clinical decision support systems.

Key Points

  • **Unified Radiomics Framework**: PySERA integrates both handcrafted radiomics (557 features, including 487 IBSI compliant) and deep learning radiomics embeddings from pre-trained networks (e.g., ResNet50, DenseNet121, VGG16) within a single, modular Python library.
  • **Enhanced Reproducibility and Standardization**: The platform achieves over 94% reproducibility in Image Biomarker Standardization Initiative (IBSI) benchmarks, outperforming PyRadiomics and closely matching MITK, by adhering to standards and providing standardized preprocessing.
  • **Automated and Scalable Feature Extraction**: PySERA incorporates standardized preprocessing (resampling, discretization, normalization), multi-format image I/O, adaptive memory management, and a parallel multi-core extraction engine for efficient and automated processing.
  • **Seamless Machine Learning Integration**: Designed with an object-oriented architecture, PySERA natively integrates with major machine learning ecosystems such as scikit-learn, PyTorch, TensorFlow, and MONAI, streamlining AI model development.
  • **Improved Predictive Performance**: Across eight public datasets, PySERA consistently demonstrated superior predictive accuracies ranging from 0.54 to 0.87, significantly exceeding the performance of PyRadiomics.
  • **Open-Source Python Native**: Reimplementing the MATLAB-based SERA platform in Python makes PySERA accessible, transparent, and extensible for the broader medical imaging and AI research communities.

Methodology

The authors developed PySERA as an open-source, Python-native reimplementation of the MATLAB-based SERA platform, utilizing a modular, object-oriented architecture. They implemented 557 handcrafted features (including IBSI compliant, moment invariant, and diagnostic) and integrated deep learning embeddings from pre-trained networks. The framework includes standardized preprocessing and a parallel extraction engine. Performance was rigorously evaluated through IBSI benchmarks for reproducibility, comparative analysis against PyRadiomics and MITK, and assessment of predictive accuracy across eight public datasets using machine learning models.

Abstract

Radiomics enables the extraction of quantitative biomarkers from medical images for precision modeling, but reproducibility and scalability remain limited due to heterogeneous software implementations and incomplete adherence to standards. Existing tools also lack unified support for deep learning based radiomics. To address these limitations, we introduce PySERA, an open source, Python native, standardized radiomics framework designed for automation, reproducibility, and seamless AI integration. PySERA reimplements the MATLAB based SERA platform within a modular, object oriented architecture and computes 557 features, including 487 Image Biomarker Standardization Initiative (IBSI) compliant features, 10 moment invariant descriptors, and 60 diagnostic features, together with deep learning radiomics embeddings from pretrained networks such as ResNet50, DenseNet121, and VGG16. The framework provides standardized preprocessing, including resampling, discretization, and normalization, multi format image input and output, adaptive memory management, and a parallel multicores extraction engine. PySERA integrates natively with major machine learning ecosystems including scikit learn, PyTorch, TensorFlow, MONAI and others. In IBSI benchmarks, PySERA achieved more than 94 percent reproducibility and outperformed PyRadiomics while closely matching MITK. Across eight public datasets, it achieved predictive accuracies ranging from 0.54 to 0.87, consistently exceeding PyRadiomics. PySERA unifies handcrafted and deep learning radiomics in a transparent, scalable, and extensible Python framework, establishing a robust foundation for reproducible, AI ready imaging research.
