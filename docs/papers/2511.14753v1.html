SparseST: Exploiting Data Sparsity in Spatiotemporal Modeling and Prediction

arXiv ID: 2511.14753v1
Published: 2025-11-18
Authors: Junfeng Wu, Hadjer Benmeziane, Kaoutar El Maghraoui, Liu Liu, Yinan Wang
Categories: cs.LG, cs.CE

Summary

This paper introduces SparseST, a novel framework designed to address the high computational cost of state-of-the-art spatiotemporal models like ConvLSTM, particularly for resource-constrained edge devices. SparseST pioneers in exploiting inherent data and feature sparsity to achieve computational efficiency while preserving model performance, a strategy largely overlooked in previous efficient AI research. The framework also features a multi-objective composite loss function to approximate the Pareto front, offering practitioners a guide to balance performance and efficiency based on specific deployment constraints.

Medical Relevance

Efficient spatiotemporal modeling is critical for real-time analysis and prediction in healthcare, especially for applications on edge devices such as wearables, remote patient monitoring systems, and smart hospital sensors, where computational resources are often limited.

AI Health Application

SparseST could enable more efficient and real-time spatiotemporal predictive models for healthcare on edge devices. This includes applications such as: predicting disease outbreaks based on geographical and temporal data, real-time analysis of multi-sensor data from wearables for early anomaly detection (e.g., fall detection, cardiac event prediction, continuous glucose monitoring), optimizing patient flow and resource allocation in hospitals, and developing low-latency AI for telemedicine or remote diagnostic tools that run directly on patient-side devices.

Key Points

  • Convolutional Long Short-Term Memory (ConvLSTM) and its variants, while highly performant in Spatiotemporal Data Mining (STDM), are computationally expensive.
  • The computational cost makes ConvLSTM models unsuitable for edge devices with limited resources, crucial for emerging edge computing applications in Complex Physical Systems (CPS) like healthcare.
  • Existing efficient AI methods primarily focus on reducing model capacity redundancy (e.g., pruning), which is less effective for STDM due to the complex dependencies requiring extensive model capacity.
  • The paper identifies and exploits a largely overlooked source of inefficiency: high levels of data and feature redundancy in spatiotemporal data.
  • SparseST is introduced as a novel framework specifically designed to leverage this data sparsity for developing efficient spatiotemporal models.
  • A multi-objective composite loss function is developed to explore and approximate the Pareto front between model performance and computational efficiency.
  • The framework provides a practical guide for practitioners, enabling them to adjust model configurations to meet specific computational resource constraints and performance requirements of downstream tasks.

Methodology

The paper proposes SparseST, a novel framework that exploits data and feature sparsity to develop computationally efficient spatiotemporal models. It incorporates a multi-objective composite loss function designed to approximate the Pareto front, allowing for the exploration and balancing of model performance against computational efficiency.

The multi-objective loss function provides a practical mechanism to explore and balance model performance with computational efficiency, approximating the Pareto front for flexible deployment based on resource constraints.

Abstract

Spatiotemporal data mining (STDM) has a wide range of applications in various complex physical systems (CPS), i.e., transportation, manufacturing, healthcare, etc. Among all the proposed methods, the Convolutional Long Short-Term Memory (ConvLSTM) has proved to be generalizable and extendable in different applications and has multiple variants achieving state-of-the-art performance in various STDM applications. However, ConvLSTM and its variants are computationally expensive, which makes them inapplicable in edge devices with limited computational resources. With the emerging need for edge computing in CPS, efficient AI is essential to reduce the computational cost while preserving the model performance. Common methods of efficient AI are developed to reduce redundancy in model capacity (i.e., model pruning, compression, etc.). However, spatiotemporal data mining naturally requires extensive model capacity, as the embedded dependencies in spatiotemporal data are complex and hard to capture, which limits the model redundancy. Instead, there is a fairly high level of data and feature redundancy that introduces an unnecessary computational burden, which has been largely overlooked in existing research. Therefore, we developed a novel framework SparseST, that pioneered in exploiting data sparsity to develop an efficient spatiotemporal model. In addition, we explore and approximate the Pareto front between model performance and computational efficiency by designing a multi-objective composite loss function, which provides a practical guide for practitioners to adjust the model according to computational resource constraints and the performance requirements of downstream tasks.
