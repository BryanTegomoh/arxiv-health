Assessing the Feasibility of Early Cancer Detection Using Routine Laboratory Data: An Evaluation of Machine Learning Approaches on an Imbalanced Dataset

arXiv ID: 2510.20209v1
Published: 2025-10-23
Authors: Shumin Li
Categories: cs.LG, cs.AI
Relevance Score: 0.95 / 1.00
View on arXiv Download PDF

Summary

This study evaluated the feasibility of early cancer detection in dogs using machine learning on routine, low-cost laboratory data from an imbalanced dataset. It revealed that while a statistically detectable cancer signal exists (AUROC = 0.815), the optimal model delivered poor clinical classification performance (F1-score = 0.25, PPV = 0.15) driven by non-specific markers, rendering it unsuitable for reliable diagnostic use in isolation. The work establishes a performance ceiling for this data modality and emphasizes the need for multi-modal data integration for future progress.

Medical Relevance

This research directly addresses the significant challenge of developing accessible, low-cost early cancer screening tools in veterinary medicine. It provides critical evidence on the inherent limitations of relying solely on routine laboratory data for early cancer detection in dogs, guiding future research toward more robust, integrated diagnostic approaches.

AI Health Application

The paper applies machine learning for early cancer detection and risk classification using routine laboratory data. It evaluates various machine learning models and techniques for diagnostic purposes, including interpretability analysis (SHAP) to understand feature importance in medical predictions. This directly falls under medical AI applications for diagnostics, risk assessment, and computational oncology.

Key Points

  • The study aimed to assess the feasibility of early cancer detection using routine, low-cost laboratory data in dogs from the Golden Retriever Lifetime Study (GRLS) cohort, accounting for real-world constraints like diverse cancer types and post-diagnosis samples.
  • A comprehensive benchmark evaluation was conducted, comparing 126 analytical pipelines incorporating various machine learning models, feature selection methods, and data balancing techniques, with patient-level data partitioning to prevent leakage.
  • The optimal model, a Logistic Regression classifier with class weighting and recursive feature elimination, achieved a moderate ranking ability (AUROC = 0.815; 95% CI: 0.793-0.836).
  • Despite moderate AUROC, the model demonstrated poor clinical classification performance, yielding an F1-score of 0.25 and a Positive Predictive Value (PPV) of 0.15.
  • A high Negative Predictive Value (NPV = 0.98) was observed, but the insufficient recall (0.79) precludes its use as a reliable rule-out test for cancer.
  • Interpretability analysis using SHapley Additive exPlanations (SHAP) indicated that predictions were primarily driven by non-specific features such as age and general markers of inflammation and anemia.
  • The study concludes that the cancer signal in routine lab data, when used in isolation, is too weak and confounded to reliably discriminate cancer from normal aging or other inflammatory conditions for clinical applications, necessitating multi-modal data integration.

Methodology

The study performed a comprehensive benchmark evaluation of 126 machine learning analytical pipelines. These pipelines systematically combined diverse ML models (e.g., Logistic Regression), feature selection methods (e.g., Recursive Feature Elimination), and data balancing techniques (e.g., class weighting) to handle the severe class imbalance. Data from the Golden Retriever Lifetime Study (GRLS) cohort were used, with patient-level partitioning to prevent data leakage. Model interpretability was assessed using SHapley Additive exPlanations (SHAP).

Key Findings

The optimal model, a Logistic Regression classifier with class weighting and recursive feature elimination, achieved a moderate AUROC of 0.815 (95% CI: 0.793-0.836), but exhibited poor clinical classification performance with an F1-score of 0.25 and a Positive Predictive Value (PPV) of 0.15. While a high Negative Predictive Value (NPV) of 0.98 was noted, the recall of 0.79 was insufficient for reliable rule-out applications. SHAP analysis revealed that model predictions were predominantly driven by non-specific features like age and markers of inflammation and anemia, indicating a weak and confounded cancer signal.

Clinical Impact

The findings demonstrate that current machine learning approaches using routine laboratory data alone are not clinically reliable for early cancer detection or as a rule-out test due to poor classification performance (low F1, low PPV) and insufficient recall despite high NPV. This establishes a critical performance ceiling for this data modality in isolation, indicating it cannot reliably differentiate early cancer from normal aging or other inflammatory conditions. Therefore, it is not suitable for practical veterinary screening or diagnostic applications in its current form.

Limitations

The primary limitation identified is that routine laboratory data, when used in isolation, provides a cancer signal that is too weak and confounded by non-specific features (such as age, inflammation, and anemia) to achieve clinically reliable discrimination. This work effectively defines the performance ceiling for this specific data modality.

Future Directions

The paper explicitly suggests that "meaningful progress in computational veterinary oncology will require integration of multi-modal data sources." This implies future research should focus on combining routine laboratory data with other information types, such as genetic data, imaging, or advanced biomarkers, to develop more accurate and clinically actionable early cancer detection tools.

Abstract

The development of accessible screening tools for early cancer detection in dogs represents a significant challenge in veterinary medicine. Routine laboratory data offer a promising, low-cost source for such tools, but their utility is hampered by the non-specificity of individual biomarkers and the severe class imbalance inherent in screening populations. This study assesses the feasibility of cancer risk classification using the Golden Retriever Lifetime Study (GRLS) cohort under real-world constraints, including the grouping of diverse cancer types and the inclusion of post-diagnosis samples. A comprehensive benchmark evaluation was conducted, systematically comparing 126 analytical pipelines that comprised various machine learning models, feature selection methods, and data balancing techniques. Data were partitioned at the patient level to prevent leakage. The optimal model, a Logistic Regression classifier with class weighting and recursive feature elimination, demonstrated moderate ranking ability (AUROC = 0.815; 95% CI: 0.793-0.836) but poor clinical classification performance (F1-score = 0.25, Positive Predictive Value = 0.15). While a high Negative Predictive Value (0.98) was achieved, insufficient recall (0.79) precludes its use as a reliable rule-out test. Interpretability analysis with SHapley Additive exPlanations (SHAP) revealed that predictions were driven by non-specific features like age and markers of inflammation and anemia. It is concluded that while a statistically detectable cancer signal exists in routine lab data, it is too weak and confounded for clinically reliable discrimination from normal aging or other inflammatory conditions. This work establishes a critical performance ceiling for this data modality in isolation and underscores that meaningful progress in computational veterinary oncology will require integration of multi-modal data sources.
