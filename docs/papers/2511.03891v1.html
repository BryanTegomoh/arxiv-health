Improving Diagnostic Performance on Small and Imbalanced Datasets Using Class-Based Input Image Composition

arXiv ID: 2511.03891v1
Published: 2025-11-05
Authors: Hlali Azzeddine, Majid Ben Yakhlef, Soulaiman El Hazzat
Categories: cs.CV, cs.AI, cs.DB
Relevance Score: 0.98 / 1.00
View on arXiv Download PDF

Summary

This paper introduces Class-Based Image Composition (CoImg), a novel deep learning approach that fuses multiple images of the same diagnostic class into composite visual inputs to enhance intra-class variance and information density. Applied to an imbalanced Optical Coherence Tomography (OCT) retinal dataset, this method significantly improved diagnostic performance. By creating a balanced dataset (Co-OCTDL) where each sample is a 3x1 composite image, the authors achieved near-perfect accuracy (99.6%) and substantially lower false prediction rates compared to a baseline model trained on raw data.

Medical Relevance

This method offers a robust solution for improving the accuracy and reliability of AI-driven diagnostics in ophthalmology and other medical fields, especially when dealing with limited or imbalanced clinical imaging data. By boosting diagnostic performance and reducing false positives/negatives, it can lead to more precise and early detection of diseases, which is crucial for timely patient intervention and improved outcomes.

AI Health Application

The AI application involves developing more robust and accurate deep learning models for automated medical diagnosis from imaging data (specifically OCT scans of the retina). By improving model performance on challenging small and imbalanced datasets, it facilitates earlier and more precise detection of various eye diseases, leading to better patient outcomes. This method can be integrated into clinical decision support systems to assist ophthalmologists in diagnosing complex or subtle conditions.

Key Points

  • Addresses the critical challenges of high false prediction rates in deep learning models due to small, imbalanced datasets and poor input image quality in medical imaging.
  • Proposes Class-Based Input Image Composition (CoImg), a technique that reformulates training inputs by fusing multiple images of the same class into combined visual composites.
  • CoImg aims to enhance intra-class variance and improve the valuable information density per training sample, enabling the model to distinguish subtle disease patterns more effectively.
  • The method was evaluated using the OCTDL dataset, comprising 2,064 high-resolution OCT scans of the human retina across seven distinct, significantly imbalanced disease classes.
  • A perfectly class-balanced version of the dataset, Co-OCTDL, was constructed where each original scan is represented as a 3x1 layout composite image (CoImg).
  • A comparative analysis using an identical VGG16 model architecture and hyperparameters demonstrated marked improvements with the Co-OCTDL representation.
  • The enhanced Co-OCTDL dataset achieved near-perfect diagnostic results: 99.6% accuracy, 0.995 F1-score, and 0.9996 AUC, alongside a significantly lower false prediction rate, validating its effectiveness for challenging datasets.

Methodology

The core methodology is Class-Based Input Image Composition (CoImg). This involves programmatically fusing multiple input images belonging to the same diagnostic class into a single composite visual representation, typically arranged in a defined layout (e.g., 3x1). For evaluation, the authors applied this to the OCTDL dataset to create Co-OCTDL, a perfectly class-balanced version where each original scan was replaced by a CoImg formed from multiple scans of its corresponding class. A VGG16 deep learning model was then trained independently on both the original OCTDL and the Co-OCTDL datasets, ensuring identical model architecture and hyperparameters for a fair comparative analysis of diagnostic performance.

Key Findings

The proposed Class-Based Input Image Composition method, when applied to create the Co-OCTDL dataset, led to a substantial improvement in diagnostic performance. Using a VGG16 model, the Co-OCTDL dataset achieved a near-perfect accuracy of 99.6%, an F1-score of 0.995, and an AUC of 0.9996. These results represent a significant enhancement over the baseline model trained on the raw OCTDL dataset, demonstrating a markedly lower false prediction rate and confirming the method's efficacy for improving predictions on small and imbalanced medical imaging datasets.

Particularly beneficial for ophthalmology, where OCT imaging is routine, it enables more accurate detection and classification of retinal diseases even when training data is scarce or unevenly distributed. Improved diagnostic accuracy and reduced false predictions can support earlier and more precise clinical decisions, optimize patient management, and potentially facilitate broader adoption of AI in diagnostics, especially in areas with limited access to specialized expertise.

Abstract

Small, imbalanced datasets and poor input image quality can lead to high false predictions rates with deep learning models. This paper introduces Class-Based Image Composition, an approach that allows us to reformulate training inputs through a fusion of multiple images of the same class into combined visual composites, named Composite Input Images (CoImg). That enhances the intra-class variance and improves the valuable information density per training sample and increases the ability of the model to distinguish between subtle disease patterns. Our method was evaluated on the Optical Coherence Tomography Dataset for Image-Based Deep Learning Methods (OCTDL) (Kulyabin et al., 2024), which contains 2,064 high-resolution optical coherence tomography (OCT) scans of the human retina, representing seven distinct diseases with a significant class imbalance. We constructed a perfectly class-balanced version of this dataset, named Co-OCTDL, where each scan is resented as a 3x1 layout composite image. To assess the effectiveness of this new representation, we conducted a comparative analysis between the original dataset and its variant using a VGG16 model. A fair comparison was ensured by utilizing the identical model architecture and hyperparameters for all experiments. The proposed approach markedly improved diagnostic results.The enhanced Dataset achieved near-perfect accuracy (99.6%) with F1-score (0.995) and AUC (0.9996), compared to a baseline model trained on raw dataset. The false prediction rate was also significantly lower, this demonstrates that the method can producehigh-quality predictions even for weak datasets affected by class imbalance or small sample size.
