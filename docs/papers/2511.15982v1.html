Machine Learning Epidemic Predictions Using Agent-based Wireless Sensor Network Models

Published: 2025-11-20
Authors: Chukwunonso Henry Nwokoye, Blessing Oluchi, Sharna Waldron, Peace Ezzeh

Summary

This paper addresses the fundamental challenge of insufficient epidemiological data in Wireless Sensor Networks (WSNs) for robust threat prediction by developing a machine learning framework for epidemic forecasting. It utilizes an agent-based implementation of the SEIRV model to generate synthetic datasets and applies various ML algorithms to predict the number of infected and recovered nodes. The study demonstrates effective predictive performance, particularly with ensemble methods, offering a methodological contribution transferable to broader epidemic modeling.

Medical Relevance

Although primarily focused on malware spread in Wireless Sensor Networks, the core methodology of using agent-based SEIRV models and machine learning for epidemic prediction is directly transferable to forecasting human disease outbreaks, assessing public health interventions, and guiding resource allocation during real-world epidemics.

AI Health Application

This research provides a framework for developing machine learning models to predict the spread of epidemics using established epidemiological models like SEIRV. The demonstrated methodology, involving agent-based simulations, synthetic data generation, and the application of ML algorithms for forecasting infection and recovery rates, can be directly applied to real-world biological epidemics. This is crucial for developing medical AI applications for early warning systems for disease outbreaks, informing public health interventions, and optimizing resource allocation in healthcare systems.

Key Points

  • Addresses the fundamental difficulty of constructing robust epidemic forecasting models in Wireless Sensor Networks (WSNs) due to a lack of epidemiological data.
  • Employed an agent-based implementation of the Susceptible-Exposed-Infected-Recovered-Vaccinated (SEIRV) mathematical model to simulate epidemic spread.
  • Generated two synthetic epidemic datasets using NetLogo's BehaviorSpace and Python, serving as the basis for machine learning predictions.
  • Framed the prediction task as a regression problem, specifically targeting the forecasting of infected and recovered nodes within the simulated network.
  • Evaluated several machine learning algorithms, comparing their performance using error metrics on train/test sets and R^2 values.
  • Achieved high R^2 values (up to 1.000 on training, 0.999 on validation), demonstrating effective model fit and predictive performance on unseen data.
  • Identified Random Forest, XGBoost, Decision Trees, and k-nearest neighbors as the best-performing algorithms, while linear models and Support Vector Regression showed poorer results.

Methodology

The study employed an agent-based implementation of the Susceptible-Exposed-Infected-Recovered-Vaccinated (SEIRV) mathematical model to simulate epidemic dynamics within a Wireless Sensor Network context. Two synthetic epidemic datasets were generated using NetLogo's BehaviorSpace and Python. Various machine learning algorithms (e.g., Random Forest, XGBoost, Decision Trees, k-nearest neighbors, Support Vector, Linear, Lasso, Ridge, ElasticNet regression) were then applied to these datasets, posing the problem as a regression task to predict the number of infected and recovered nodes. Algorithm performance was rigorously evaluated using error metrics on train and test sets, and R^2 values.

Key Findings

Machine learning models, particularly ensemble methods, demonstrated high efficacy in predicting the number of infected and recovered nodes in simulated epidemics, achieving R^2 values up to 0.999 on validation data. Specifically, Random Forest, XGBoost, Decision Trees, and k-nearest neighbors significantly outperformed linear regression models (Linear, Lasso, Ridge, ElasticNet) and Support Vector Regression, which were identified as the worst-performing algorithms.

Clinical Impact

This research offers a robust computational framework that, when adapted to biological contexts, could significantly enhance public health capabilities for more accurate and proactive prediction of infectious disease outbreaks. Such predictive models could aid in establishing early warning systems, optimizing resource allocation for medical supplies and personnel, and evaluating the potential effectiveness of public health interventions like vaccination campaigns or social distancing measures.

Limitations

A primary limitation is the reliance on synthetic epidemic datasets generated from Wireless Sensor Network (WSN) models rather than real-world human epidemiological data. This may restrict the direct generalizability of the findings to biological epidemics without further validation against complex, real-world population dynamics and disease specificities.

Future Directions

The abstract does not explicitly state future research directions. However, logical next steps would include validating this methodology with real-world human epidemiological data, incorporating more complex socio-demographic and environmental factors into the agent-based models, and exploring the integration of these predictive models into actionable public health decision-support systems for real-time epidemic management.

Abstract

The lack of epidemiological data in wireless sensor networks (WSNs) is a fundamental difficulty in constructing robust models to forecast and mitigate threats such as viruses and worms. Many studies have examined different epidemic models for WSNs, focusing on how malware infections spread given the network's specific properties, including energy limits and node mobility. In this study, an agent-based implementation of the susceptible-exposed-infected-recovered-vaccinated (SEIRV) mathematical model was employed for machine learning (ML) predictions. Using tools such as NetLogo's BehaviorSpace and Python, two epidemic synthetic datasets were generated and prepared for the application of several ML algorithms. Posed as a regression problem, the infected and recovered nodes were predicted, and the performance of these algorithms is compared using the error metrics of the train and test sets. The predictions performed well, with low error metrics and high R^2 values (0.997, 1.000, 0.999, 1.000), indicating an effective fit to the training set. The validation values were lower (0.992, 0.998, 0.971, and 0.999), as is typical when evaluating model performance on unseen data. Based on the recorded performances, support vector, linear, Lasso, Ridge, and ElasticNet regression were among the worst-performing algorithms, while Random Forest, XGBoost, Decision Trees, and k-nearest neighbors achieved the best results.
