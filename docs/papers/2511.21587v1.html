Summary

This tutorial demystifies Approximate Bayesian Computation with Sequential Monte Carlo (ABC-SMC), a powerful likelihood-free method for parameter inference in mechanistic models with intractable likelihoods. Using Python's PyMC library and practical examples from predator-prey to hierarchical epidemic dynamics, it guides researchers through implementing, diagnosing, and interpreting ABC-SMC analyses, bridging the gap for those hesitant to adopt these complex methods.

Medical Relevance

This tutorial is highly relevant to medicine and health by enabling researchers to robustly infer parameters for complex mechanistic models crucial in epidemiology, infectious disease research, and other life sciences, which is essential for understanding disease dynamics, predicting outbreaks, and optimizing public health interventions.

AI Health Application

The paper provides a practical guide to ABC-SMC using PyMC for parameter inference in mechanistic models, specifically applied to 'hierarchical epidemic models.' This constitutes an AI/ML application (probabilistic programming for advanced statistical inference) that supports data-driven decision-making, prediction, and understanding of disease dynamics critical for public health, medical intervention strategies, and biosecurity assessments.

Methodology

The paper presents a tutorial on implementing Approximate Bayesian Computation with Sequential Monte Carlo (ABC-SMC), a likelihood-free inference method. It demonstrates how to infer parameters for mechanistic dynamical systems, requiring only the ability to simulate data. The implementation uses Python's `pymc` probabilistic programming framework, providing practical code examples that range from basic ecological models to complex hierarchical epidemic models, guiding users through the entire workflow of implementation, diagnosis, and interpretation.

Key Findings

The tutorial successfully demonstrates that ABC-SMC, when taught through practical, PyMC-based examples, can effectively overcome its perceived complexity, making it accessible for researchers. It illustrates how to practically apply ABC-SMC to infer parameters in challenging dynamical mechanistic models, including hierarchical epidemic models, even in scenarios with intractable likelihoods and partial observability, thereby empowering users to understand, implement, and interpret these advanced Bayesian analyses.

This capability can lead to more accurate predictions of disease trajectories, better-informed public health policy decisions, more effective resource allocation during epidemics, and ultimately improved patient care strategies.

Abstract

Mechanistic models are essential tools across ecology, epidemiology, and the life sciences, but parameter inference remains challenging when likelihood functions are intractable. Approximate Bayesian Computation with Sequential Monte Carlo (ABC-SMC) offers a powerful likelihood-free alternative that requires only the ability to simulate data from mechanistic models. Despite its potential, many researchers remain hesitant to adopt these methods due to perceived complexity. This tutorial bridges that gap by providing a practical, example-driven introduction to ABC-SMC using Python. From predator-prey dynamics to hierarchical epidemic models, we illustrate by example how to implement, diagnose, and interpret ABC-SMC analyses. Each example builds intuition about when and why ABC-SMC works, how partial observability affects parameter identifiability, and how hierarchical structures naturally emerge in Bayesian frameworks. All code leverages PyMC's modern probabilistic programming interface, ensuring reproducibility and easy adaptation to new problems. The code its fully available for download at \href{https://github.com/mariocastro73/ABCSMC_pymc_by_example}{mariocastro73/ABCSMC\_pymc\_by\_example}
