Summary

Approximate Bayesian Computation with Sequential Monte Carlo (ABC-SMC) offers a powerful likelihood-free method for parameter inference in complex mechanistic models common in ecology, epidemiology, and life sciences where traditional likelihood functions are intractable. This tutorial provides a practical, example-driven introduction to implementing, diagnosing, and interpreting ABC-SMC analyses using Python and PyMC, addressing perceived adoption complexities for researchers.

Medical Relevance

This methodology is highly relevant for epidemiological modeling, personalized medicine, and drug development, enabling robust parameter inference for complex disease progression, treatment response, or pathogen transmission models where explicit likelihoods are often unavailable. It helps researchers move beyond purely statistical models to more mechanistic understandings crucial for intervention design and evaluation.

The paper enables the application of advanced probabilistic programming techniques (ABC-SMC using PyMC) for parameter inference in complex mechanistic models, specifically including 'hierarchical epidemic models'. This constitutes a form of AI/computational modeling in health, allowing for more robust and data-driven analysis of disease spread, prediction of outbreaks, and evaluation of interventions, which is critical for public health decision-making and preparedness.

Methodology

The paper presents an instructional guide on Approximate Bayesian Computation with Sequential Monte Carlo (ABC-SMC). The methodology involves an example-driven, step-by-step approach to implementing and applying ABC-SMC. It leverages Python with PyMC's probabilistic programming interface to simulate data from mechanistic models, compare simulated data to observed data using summary statistics, and iteratively refine parameter posteriors without explicit likelihood calculation. The tutorial also focuses on diagnostics and interpretation of the results to ensure robust inference.

Abstract

Mechanistic models are essential tools across ecology, epidemiology, and the life sciences, but parameter inference remains challenging when likelihood functions are intractable. Approximate Bayesian Computation with Sequential Monte Carlo (ABC-SMC) offers a powerful likelihood-free alternative that requires only the ability to simulate data from mechanistic models. Despite its potential, many researchers remain hesitant to adopt these methods due to perceived complexity. This tutorial bridges that gap by providing a practical, example-driven introduction to ABC-SMC using Python. From predator-prey dynamics to hierarchical epidemic models, we illustrate by example how to implement, diagnose, and interpret ABC-SMC analyses. Each example builds intuition about when and why ABC-SMC works, how partial observability affects parameter identifiability, and how hierarchical structures naturally emerge in Bayesian frameworks. All code leverages PyMC's modern probabilistic programming interface, ensuring reproducibility and easy adaptation to new problems. The code its fully available for download at \href{https://github.com/mariocastro73/ABCSMC_pymc_by_example}{mariocastro73/ABCSMC\_pymc\_by\_example}
